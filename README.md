# Clojure Calculator

It's nice to have **lein**: http://leiningen.org/

Run:
```
lein run
```

Test:
```
lein test
```

Actually there's nothing to run. All the functionality is visible in tests.

Math functions implemented:
 - *abs*
 - *power*
 - *round*
 - *sqrt*
 - *factorial*
 - *sin*
 - *con*
 - *exp*
 - *ln*
 - *binomial*
